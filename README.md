Вкладка "Products":
    Список товаров:
        При клике на "VIEW" появляется модальное окно с подробным описанием товара
        При клике на "TO CART" товар летит в корзину (см. Вкладка "CART")

Вкладка "CART":
    Корзина:
        При клике на "DELETE FROM CART" товар удаляется из корзины
        При клике на "VIEW" появляется модальное окно с подробным описанием товара
